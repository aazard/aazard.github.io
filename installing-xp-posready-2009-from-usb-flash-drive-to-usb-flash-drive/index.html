    Installing XP POSReady 2009 From USB Flash Drive - To USB Flash Drive

    Installing POSReady to a USB Flash Drive

    You can install POSReady to a USB flash drive, which makes it possible to run POSReady from the drive. There are some requirements that you should be aware of when you install POSReady to a USB flash drive:

    • You cannot install to USB flash drive partitions that are protected with a password.
    • The target USB flash drive must be assigned the drive letter C. If the drive letter is changed, unexpected application behavior may result.
    • Each computer manages USB devices differently. If you are installing POSReady to a USB flash drive from another USB flash drive, you may have to disconnect and reconnect the drives to make sure that your computer restarts from the correct drive. More information about this scenario is included in the following procedure.

    How to install POSReady from a USB flash drive to a USB flash drive

    1. Insert the source USB flash drive that contains the POSReady Setup files, and then insert your destination USB flash drive.

    2. Start the computer and make sure that the BIOS starts up from the source USB flash drive instead of another drive, such as the hard disk drive, optical drive, or the destination USB flash drive. For more information about startup order, see Configuring Your Computer to Detect a USB Flash Drive. Windows PE 2005 environment starts with the message Starting Installation Environment… and will progress to a Windows XP loading screen.

    3. POSReady Setup starts. You can continue with an Interactive Installation or an Unattended Installation. On the Storage Configuration page, make sure that you select the partition on the destination USB flash drive. As soon as installation is complete, your computer will automatically restart.

    4. Immediately after your computer restarts, disconnect the source USB flash drive from the computer to prevent restarting from the drive. POSReady will automatically start from the destination USB flash drive and start the First Boot Agent (FBA).

    5. When FBA starts, insert the source USB flash drive again. FBA requires additional POSReady Setup files on the source USB flash drive to complete the installation on the destination USB flash drive. FBA will continue automatically, and will restart your computer when installation is complete.

    6. Immediately after your computer restarts for the second time, you must disconnect the source USB flash drive from the computer to prevent starting up from the drive. Your computer will restart from the destination USB flash drive and a Microsoft Windows XP loading screen will appear. POSReady will start for the first time.

    Installing Mass Storage Drivers from a USB Flash Drive

    In POSReady you can install mass storage drivers from a USB flash drive. After completing the steps below your mass storage drivers will be available for use during setup.

     Note

    Installing mass storage drivers from a USB flash drive is only supported when you run setup from the same USB flash drive. If you choose to launch setup using the product installation disk, you will not be able to retrieve mass storage drivers from a USB drive.

    How to install mass storage drivers from a USB flash drive

    1. Connect your USB flash drive to a USB port and insert the POSReady installation disc in your computer. The POSReady Companion Setup is displayed.

    2. Choose Create USB Flash Drive Install. The Install from USB Flash Drive page is displayed.

    3. Choose your desired USB flash drive, languages, and applications.

       Note

      All data on your destination USB flash drive will be erased.

    4. Choose Install. Your desired files will be copied on the USB flash drive and the drive can now be used as startup POSReady installation media.

    5. In the root of the USB media, create a new folder named Drivers.

    6. Copy your mass storage drivers to the Drivers directory.
